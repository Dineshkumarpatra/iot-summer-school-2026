# Ultrasonic Parking Sensor

## Sensor

HC-SR04 Ultrasonic Sensor

## Features

- Measures distance in centimeters
- SAFE indication above 50 cm
- Yellow warning from 20–50 cm
- Red warning from 10–20 cm
- Flashing LEDs and continuous buzzer below 10 cm

## Formula

distance = (duration × 0.034) / 2
