# Project Description

This repository contains all assignments, hands-on experiments, and mini projects completed during the **IIT Jammu Summer School 2026 – IoT & Embedded Systems Program**.

The repository demonstrates practical implementation of embedded systems concepts using **Arduino UNO**, various sensors, actuators, and communication modules. Every assignment has been developed following proper **Git version control practices**, including feature-based commits, branching strategies, documentation, and project organization.

The assignments cover a wide range of Internet of Things (IoT) concepts, including Arduino programming, sensor interfacing, actuator control, data acquisition, wireless communication, automation, and embedded system design. Each project includes source code, circuit diagrams, Serial Monitor outputs, Git commit history, and detailed documentation.
